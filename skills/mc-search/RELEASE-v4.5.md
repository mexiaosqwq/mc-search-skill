## 主要变更

### 1. Clean Code 重构 ✨

**函数拆分**:
- 3 个大函数拆分为 **17 个小函数**
- 最大函数行数: **220 → 157 行** (↓29%)
- 函数平均行数: **100 → 30** (↓70%)
- 新增辅助函数: `_extract_wiki_infobox`, `_fuse_results` 等拆分为多个子函数

**常量提取**:
- 提取 **17 个评分和配置常量**
- 消除所有魔法数字
- 常量总数: 20 → **37 个** (↑85%)

**代码简化**:
- 嵌套层级: **4 层 → 2 层**
- 消除重复代码: **~200 行**

### 2. 硬编码清理 🧹

**项目类型常量**:
```python
_CONTENT_TYPES = {"mod", "item", "modpack", "shader", ...}
_TEXT_CONTENT_TYPES = {"mod", "item", "modpack"}
_VISUAL_CONTENT_TYPES = {"shader", "resourcepack"}
_WIKI_ONLY_TYPES = {"entity", "biome", "dimension"}
```

**类型检查优化**:
- 从硬编码字符串 → 使用常量集合
- 减少拼写错误风险

**Wiki 类型推断简化**:
- 移除 **150+ 行**硬编码关键词映射
- 简化为 **10 行**简洁逻辑
- 避免不可靠的猜测

### 3. 数据完整性增强 📊

**Modrinth 搜索字段**:
- 新增: `downloads`, `follows`, `icon_url`, `author`, `versions`
- 搜索结果返回完整统计信息

**懒加载图片支持**:
- 支持 3 种属性: `data-src`, `data-lazy-src`, `data-original`
- 更全面地捕获截图

**Bug 修复**:
1. 生物群系类型判断错误
2. 中文正则不兼容
3. CLI 重复函数定义

### 4. 开发者文档 📖

**新增 CLAUDE.md**:
- 项目架构说明
- 常用命令指南
- 开发规范
- 代码质量改进记录
